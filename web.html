WEB
New Zealand Tourism

This project was created at my time at Yoobee School of Design. The brief was the create a single page web application which gives feedback to the user about appropriate travel costs depending the travel route, the number of people in your party, the days of hire and vehicle option. The application shows fuel cost, hire cost, total distance and approximate travel time.

Multiple javascript plugins were used including mixitup.js, scrollme.js and swiper.js. Javascript was used to manipulate the DOM and show travel details to the user. Live validation was implemented into the form, which provides users with meaningful feedback when validation has passed or failed. Google maps API was used to pull data and show appropriate details to the user.

Flick Electric

Flick Electric Home Automation system was created as part of a project at my time studying towards a Diploma of Web Development at Yoobee School of Design. BANG!

BANG is a childrens picture book which was recreated for a tablet device using animated gifs that can be interacted with using Javascript & jQuery. Users are able to have a more interactive experience with simple swipe and tap functions with the use of jQuery plugins.

All the page transitions were created using jQuery mobile. The animations were created in Adobe Photoshop and were exported as gifs. BANG was created at my time at Victoria University, School of Design.
