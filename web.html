Flick Electric

Flick Electric Home Automation system was created as part of a project at my time studying towards a Diploma of Web Development at Yoobee School of Design. The brief was to design a Home Automation system for an existing company that was responsive for mobile, tablet and smart tv. The front-end was created using CSS Media Queries, jQuery scripting language & Bootstrap Framework.
